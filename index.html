O que são Variaveis CSS?

São entidades usadas para armazenar valores específicos que serão usados mais vezes.

Versionamento de código

O versionamento de código é a separação entre as alterações de um código, usada para organizar e proteger as versões anteriores.

o que é commit - comentar sobre controle de versão?

É a realização permanente de um conjunto de alterações que estão sendo feitas no projeto naquele instante. O controle de versão é um sistema que adiciona as mudanças ao repositório.

O que é um deploy em programação?

É o processo de publicar um site ou sincronizar as alterações feitas no seu código-fonte.
